🎓 Öğrenci Otomasyonu PRO | v7.1
Modern arayüzlü, yerel veritabanı destekli ve kurumsal raporlama yeteneklerine sahip profesyonel bir yönetim yazılımıdır. Geliştirici hesabı gerektirmeyen Google Auth simülasyonu ile eşsiz bir kullanıcı deneyimi sunar.

🚀 Öne Çıkan Özellikler
🛡️ SQLite Veritabanı: Veriler kalıcı olarak otomasyon_ctk.db dosyasında saklanır.

🌐 Akıllı Google Login: Tarayıcı üzerinden otomatik yönlendirme ve şifresiz giriş motoru.

🔑 Rol Bazlı Erişim: Admin ve Standart Kullanıcı modları ile tam güvenlik.

📊 Excel Raporlama: xlsxwriter ile zebra desenli, filtreli ve kurumsal rapor çıktısı.

🔍 Dinamik Arama: Binlerce kayıt arasından anlık (real-time) filtreleme.

🎨 Modern UI: CustomTkinter ile pürüzsüz, göz yormayan Dark Mode tasarımı.

🛠️ Kurulum ve Hazırlık
Projeyi çalıştırmak için sisteminizde aşağıdaki kütüphanelerin yüklü olduğundan emin olun:

Bash
pip install customtkinter pandas xlsxwriter
