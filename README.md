# Excel
.

🎓 Öğrenci Otomasyonu PRO
Modern arayüzlü, SQLite veritabanı destekli ve Google Login simülasyonuna sahip profesyonel bir öğrenci kayıt ve yönetim sistemidir. CustomTkinter kütüphanesi kullanılarak geliştirilmiş, sürüm hatalarından arındırılmış ve tamamen stabil bir masaüstü yazılımıdır.

✨ Öne Çıkan Özellikler
🛡️ Gelişmiş Giriş Sistemi: Klasörde saklanan otomasyon_ctk.db dosyası ile kalıcı veritabanı.

🌐 Google ile Giriş: Şifre girmeden, yerel sunucu üzerinden otomatik doğrulama sağlayan Google Login simülasyonu.

🔑 Yetki Yönetimi: * Admin: Kayıt ekleme, silme, güncelleme ve kullanıcı yetkilerini (Admin/User) yönetme hakkı.

User: Sadece verileri görüntüleme ve profesyonel Excel raporu alma hakkı.

📊 Zengin Excel Çıktısı: xlsxwriter motoru ile zebra desenli, filtreli ve başlıklı kurumsal Excel raporları.

🔍 Anlık Arama: Binlerce kayıt arasından isim veya okul numarası ile saniyeler içinde filtreleme.

🛠️ Kurulum
Projeyi çalıştırmak için bilgisayarınızda Python yüklü olmalı ve aşağıdaki kütüphaneleri kurmalısınız:

Bash
pip install customtkinter pandas xlsxwriter
🚀 Çalıştırma
Terminal veya komut satırına şu komutu yazarak sistemi başlatabilirsiniz:

Bash
python script.py
